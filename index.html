CH.01
SilentReading: EEG to Text
09/2025 – 06/2026 · Graduation Project

An AI system that translates EEG brainwaves into readable text for speech-impaired individuals. Built a Multi-Stream Transformer with contrastive pretraining to map biological signals to language, then fine-tuned a BART-large decoder using a chain-thaw strategy. A regex + GPT cleanup module removes decoding artifacts before the final output.

SHEFAA: Arabic Medical Question Answering
